***PROJECT***
TypiCodeAPI

***DESCRIPTION***
This is a sample API test project using RestAssured, testNG, Java, Maven and
IntelliJ idea.

It contains positive and negative scenarios(see comments in class) in two
different classes and one class configured for Parallel testing using 3
threads.


For more info regarding the mock API, consult:
https://jsonplaceholder.typicode.com/
